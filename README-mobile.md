# 手机端访问说明

1. 启动后端：

```powershell
cd "e:\5.29测试\backend"
npm run dev
```

2. 启动前端（允许局域网访问）：

```powershell
cd "e:\5.29测试\frontend"
npm run dev -- --host 0.0.0.0
```

3. 在电脑上查看本机局域网 IP（例如 `192.168.1.23`）。

4. 在手机浏览器中打开：

```text
http://192.168.1.23:5173
```

5. 如果手机打不开，请检查：
- 手机和电脑是否在同一个 Wi-Fi
- Windows 防火墙是否拦截 5173 / 8080
- 后端是否仍在运行
- 前端是否真的以 `--host 0.0.0.0` 启动
