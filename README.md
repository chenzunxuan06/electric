# 面向能源可信数据空间的多方安全协同与隐私保护技术创新解决方案

## 一、项目简介
本项目面向新型电力系统与能源可信数据空间建设需求，构建一套集“可信身份、权限控制、链上存证、隐私计算、智能问答、审计追溯、财务测算、安全态势展示”于一体的综合平台。

系统重点解决能源数据跨主体流通中的真实性校验、授权控制、隐私保护、审计追踪与智能协同问题，适用于新能源发电数据、用户用电数据、电力交易数据等场景。

---

## 二、项目核心功能

### 1. 登录与权限控制
- 支持账号密码登录
- JWT 令牌鉴权
- 角色分层：管理员、业务负责人、数据分析员

### 2. 业务首页
- 山东电网风格首页展示
- 数据资产总览
- 区块链存证记录
- 授权申请与审计日志
- 隐私计算任务
- 智能问答
- 财务测算与风控

### 3. 安全态势大屏
- 在线节点统计
- 链上交易统计
- 风险告警趋势
- 安全事件分类图
- 审计摘要展示

### 4. 权限管理
- 角色能力展示
- 菜单与权限示例
- 可扩展用户管理与审批管理

### 5. 技术说明
- 前端：Vue 3 + Vite + Element Plus + ECharts
- 后端：Express + JWT + CORS + Morgan
- 适配：PC 与移动端响应式布局
- 数据：前端模拟数据 + 后端统一接口

---

## 三、系统架构说明

### 1. 前端层
负责界面展示、交互操作、图表渲染、表格展示和响应式布局。

### 2. 接口层
后端提供统一 REST API，用于登录、统计、资产、存证、授权、审计、隐私计算、财务测算与问答。

### 3. 业务模拟层
以 mock 数据模拟能源可信数据空间场景中的关键能力，便于比赛展示与原型验证。

### 4. 扩展层
可进一步接入：
- 联盟链存证系统
- 隐私计算引擎
- DID 数字身份
- 国密算法服务
- 大模型服务

## 五、如何运行项目

### 1. 启动后端
进入 `backend` 目录执行：

```bash
npm install
npm run dev
```

默认启动在：`http://localhost:8080`

### 2. 启动前端
进入 `frontend` 目录执行：

```bash
npm install
npm run dev
```

默认启动在：`http://localhost:5173`

### 3. 打开页面
浏览器访问：

```text
http://localhost:5173
```

默认账号：
- 用户名：`admin`
- 密码：`123456`
